# Code style & conventions

- TypeScript, ESM imports with `.js` extensions in TS source.
- Prettier: singleQuote true, semi true, printWidth 100, trailingComma all.
- ESLint: `typescript-eslint` recommended type-checked; no-floating-promises/error, no-misused-promises/error, consistent-type-imports warn.
- Async/await heavy; prefer typed objects and explicit options.
